# automacao-qualidade-python
Automação de validação de dados financeiros e estoque com Python e testes automatizados (Unittest).
# Automação de Qualidade e Segurança (Python)

## 1. O Problema
Sistemas operacionais e financeiros são vulneráveis a erros humanos, como entradas de dados inválidas ou falhas de cálculo (ZeroDivisionError), comprometendo a integridade dos dados corporativos.

## 2. A Solução
Desenvolvimento de uma suíte de automação em Python que implementa camadas de validação e tratamento de erros. O sistema garante que dados incorretos sejam filtrados antes do processamento, mantendo a estabilidade do sistema.

## 3. Tecnologias Utilizadas
- **Linguagem:** Python (Clean Code, PEP 8)
- **Qualidade:** Unittest (Testes automatizados)
- **Conceitos:** Programação defensiva e validação de dados.

## 4. Diferenciais Técnicos
* **Resiliência:** Tratamento rigoroso de exceções para evitar que o sistema pare por erros inesperados.
* **Qualidade (TDD):** Uso de testes unitários para validar múltiplos cenários antes da execução final.

## 5. Como rodar
1. Clone o repositório: `git clone https://github.com/marianarochaads/automacao-qualidade-python.git`
2. Execute os testes: `python -m unittest [nome_do_arquivo].py`
